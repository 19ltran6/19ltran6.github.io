Bat Catch

Bat Catch is an easy game that features the player as a bat. The objective of the game is simple, just catch the butterflies to keep the bat from starving!


1. Describe some programming concepts or structures you learned in this unit that you applied in your game.

I learned that programming a game is very similar to programming a website through html. The codes have to be specific and no matter what happens, the computer will try
its best to run through the code until the end. This might come with some conflict but it can be easily fixed. I also noticed that numbers play an important role
in the programming. For example, it was important to note the values of the x and y axes in order to program where the butterflies would go, and be careful of using the data variable.


2. Did you encounter any problems or struggles while working on this project? If you did, how did you solve or overcome them?

I had troubles understanding the code and made many mistakes that involved codes that conflicted. I resolved theses problems by going over the entire code to understand
each code's purpose and finally found the conflicting codes.


3. What aspects of your game do you like best?

I liked how the game incorporated movement for both the player and the game sprites. It made the game interactive and enjoyable.


4. If more time was given for you to work on your game, what would you improve or modify about your game?

I would have liked to fix the buggy movement of the butterflies and include sounds to make the game more interactive.
